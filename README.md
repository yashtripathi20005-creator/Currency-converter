# 💱 Currency Converter

A Python-based currency converter application that uses the Exchange Rate API to fetch real-time exchange rates.

## ✨ Features

- Real-time exchange rates via API
- Cache system to reduce API calls
- Support for over 160 currencies
- Interactive command-line interface
- Cross-currency conversion
- Configurable base currency
- Currency information display

## 📋 Prerequisites

- Python 3.7 or higher
- Internet connection for API calls
- Free API key from [Exchange Rate API](https://exchangerate-api.com)
